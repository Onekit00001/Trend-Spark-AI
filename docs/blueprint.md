# **App Name**: TrendSpark AI

## Core Features:

- Content Idea Generation: Generate content ideas based on niche and platform selections. Employs PHP array_rand to select 5 unique ideas from a pool of 10,000+ entries, with ideas including a Hook, Format, 5 Hashtags, CTA, 5-step elaboration, and text flowchart.
- Hashtag Generation: Generate 15 random hashtags from a hard-coded set of 500+ hashtag groups based on the input niche, to help increase content visibility.
- Caption Template Generation: Generate a full caption with emoji, line-breaks, and a CTA based on the chosen platform and niche. Uses a hard-coded pool of caption templates.
- Dynamic Ad Placement: Display strategically positioned Google AdSense ads, with responsive leaderboard in the header, sticky sidebar rectangle, and in-feed ads tailored for different devices.

## Style Guidelines:

- Primary color: A vibrant shade of blue (#3B82F6), offering a sense of trust and innovation suitable for a content-creation application.
- Background color: Light gray (#F9FAFB), providing a clean and modern aesthetic that emphasizes content presentation.
- Accent color: A calming shade of green (#6EE7B7) for CTAs and important interactive elements, inspired by growth.
- Font: 'Inter' (sans-serif) for all text elements. Note: currently only Google Fonts are supported.
- Mobile-first design approach to ensure optimal viewing experience on various devices.
- Rounded cards and smooth hover shadows for an engaging and modern user interface.
